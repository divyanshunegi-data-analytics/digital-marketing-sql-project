# 📊 Digital Marketing Campaign Performance Analysis (SQL Project)

## 📌 Project Overview

This project analyzes digital marketing campaign performance using SQL.

## 📂 Dataset

The dataset includes:

* Platform (Google Ads, Facebook Ads, etc.)
* Impressions, Clicks, Conversions
* Ad Spend, Revenue, ROAS

## 🎯 Objectives

* Identify best performing platforms
* Analyze campaign performance
* Evaluate ROI and conversions

## 🛠️ Tools Used

* MySQL

## 📈 Key Analysis

* Revenue by platform
* Conversion rate analysis
* Campaign performance comparison

## 📁 Files

* dataset.csv → raw dataset
* analysis_queries.sql → SQL queries
